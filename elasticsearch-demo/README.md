## Elasticsearch Demo
> 使用Spring Data Elasticsearch完成对Elasticsearch的基本操作

包括：

- 索引库创建以及mapping的创建
- 数据的增删改查
- 基于`NativeSearchQueryBuilder`的原生查询
- 基于`NativeSearchQueryBuilder`的聚合查询
